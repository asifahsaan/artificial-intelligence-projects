# MedScan Mentor

## Student Name
Azmat Ullah

## Project Title
Medscan Mentor

## Problem Statement
MedScan Mentor is an innovative educational platform designed to assist medical students in detecting and analyzing brain diseases through CT scans. Our platform leverages advanced machine learning models and state-of-the-art technology to mentor students, providing hands-on experience and expert guidance in the field of neuroimaging.

## Dataset
MRI for Brain Tumor with Bounding Boxes  
https://www.kaggle.com/datasets/ahmedsorour1/mri-for-brain-tumor-with-bounding-boxes?resource=download

## Model / Technique Used
- Convolutional Neural Network (CNN)
- Image Classification
- Python

## How to Run the Project
1. Open `train_model.ipynb` → run all cells to train model  
2. Open `test_model.ipynb` → run on test images  

## Results
The model successfully identifies tumor regions in brain scans.
